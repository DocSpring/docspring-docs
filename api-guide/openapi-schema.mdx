We define our API endpoints and request/response schemas using the [OpenAPI specification](https://swagger.io/docs/specification/about/) (formerly known as [Swagger](https://swagger.io/)).

<LinkCard
  title="OpenAPI Schema (openapi.json)"
  href="https://docspring.com/api-docs/v1/openapi.json"
  description="View DocSpring's OpenAPI schema"
/>

We use this schema to validate your API requests and ensure that you are sending the correct properties
and data types. Our OpenAPI schema is also used to generate our [API client libraries](/docs/api-guide/autogenerated-api-clients/)
and [Postman collection](/docs/api-guide/postman/).
